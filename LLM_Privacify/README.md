# LLM_Privacify

This module analyzes privacy policies using an LLM-based pipeline. It uses a local model endpoint through LM Studio and processes privacy policy links provided in an input text file.

## Setup
1. Create and activate the Conda environment
2. Install dependencies from `requirements.txt`
3. Configure LM Studio with the required local model endpoint
4. Set the input file path for privacy policy links
5. Run the scraper/analysis script

## Main files
- `ppaf_scraper.py`
- `requirements.txt`
- `INSTALLATION.md`

## Input
A text file containing app names and privacy policy links.

## Output
Structured privacy policy analysis results for data collection and data sharing.
